# Documentation

This folder contains all project documentation.

## Structure

| Folder | Purpose | Maintained By |
|--------|---------|---------------|
| `specs/` | SDD specifications (Discovery, Requirements, Architecture, Tasks) | Spec Writer agent |
| `dev/` | Ongoing development tracking (Roadmap, Changelog, Issues) | Scribe agent |
| `guides/` | User and developer guides | Manual / Worker agent |

## Quick Links

- [Project Specifications](specs/)
- [Development Tracking](dev/)
- [User Guides](guides/)

## For AI Agents

- **Scribe**: Update `dev/` folder when features complete, bugs found/fixed, milestones reached
- **Spec Writer**: Create and maintain `specs/` folder during SDD workflow
- **Worker**: May update `guides/` when implementing user-facing features
